Dixy-lang instruction set:

          
    
            
  • 0 - increment data pointer
    • 
            
  • F - decrement data pointer
    • 
            
  • 1 - increment value of byte at data pointer
    • 
            
  • E - decrement value of byte at data pointer
    • 
            
  • 2 - output byte at data pointer to standard output device
    • 
            
  • D - accept byte of input from standard input device, storing value at data pointer
    • 
            
  • 3 - if byte at data pointer is zero, move instruction pointer to after next 'C' instruction
    • 
            
  • C - if byte at data pointer is not zero, move instruction pointer to after last '3' instruction
    • 
            
  • 4 - multiply value of X storage by byte at data pointer, storing product in Y storage
    • 
            
  • B - divide value of X storage by byte at data pointer, storing quotient in Y storage
    • 
            
  • 5 - copy value of byte at data pointer into X storage
    • 
            
  • A - copy value of X storage into byte at data pointer
    • 
            
  • 6 - copy value of byte at data pointer into Y storage
    • 
            
  • 9 - copy value of Y storage into byte at data pointer
    • 
            
  • 7 - add value of X storage and byte at data pointer, storing sum in Y storage
    • 
            
  • 8 - subtract value of X from byte at data pointer, storing difference in Y storage
